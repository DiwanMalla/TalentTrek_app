### To Get Started:

1. **Clone the Repository:**

   ```bash
   git clone https://github.com/DiwanMalla/TalentTrek_app

   ```

2. **Navigate to the Project Directory:**

   ```bash
   cd TalentTrek_app
   ```

3. **Install Dependencies:**

   ```bash
     npm install
   ```
